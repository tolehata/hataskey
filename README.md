# Hataskey

**旗鯖向けのカスタマイズCherryPickフォークです。他サーバーでの使用は想定していません。**

## 概要

Hataskey は、[CherryPick](https://github.com/kokonect-link/cherrypick)（Misskeyフォーク）をベースに、旗鯖向けの独自機能を多数追加したフォークです。当初は CherryPick-Hata として開発されていましたが、hata-11.3 より「Hataskey」を正式なブランド名としています。

- **ベースバージョン**: Misskey 2026.5.4（CherryPick 4.17.0 ベース）
- **最新リリース**: hata-11.3
- **インスタンス**:
  - [旗池2丁目](https://misskey.hatachanoima.net/)
  - [旗池3丁目](https://o.hata.blog/)
- **チェンジログ（Web）**: [home.tolehata.net/changelog](https://home.tolehata.net/changelog)
- **チェンジログ（リポジトリ）**: [HATA-CHANGELOG.md](./HATA-CHANGELOG.md)

## 主な独自機能

### Hatask（オールインワンダッシュボード）
カレンダー・ToDo・きもち記録・お花育成・Hatask Eyeを統合したダッシュボード

### 外部TL連携
外部タイムライン表示、リアクション同期、外部TL専用絵文字ピッカー、投稿・リプライ・引用対応

### Hatasaba UI
スマホ操作に特化した軽量・高速インターフェース。左右スワイプでのTL切り替えに対応

### お絵かきツール
ペン・消しゴム・図形・ぼかし・レイヤー・フィルターなど多機能なドローイングツール

### その他
- HATA CARD MAKER
- ログイン日数・ランキング機能
- タイムラインアニメーション
- 登録申請システム
- リアクション非表示機能
- ミュートユーザーリアクション非表示機能
- ゲーム機能（つみつみタワー・絵文字叩き・エモジシュート・AI対戦・サーバー対戦）
- hata-docs（機能解説ページ）
